【セットアップ手順】

① 必要なものをインストール

以下のツールを事前にインストールしてください
Docker
Docker Desktop
→ Dockerを簡単に使えるアプリケーション。WindowsやMacで必須です。
Node.js
→ フロントエンドとバックエンドの依存関係を管理します。

＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊

②フロントエンドのセットアップ

frontendフォルダに移動して、必要なパッケージをインストール

cd frontend
npm install

＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊

③バックエンドのセットアップ

frontendフォルダから、backendフォルダに移動して、必要なパッケージをインストール
（frontendフォルダから移動する場合は以下のコマンド）

cd ../backend
npm install

＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊

④Dockerで起動

※前提として、DockerDesktopは立ち上がっている状態。
プロジェクトルートに戻って以下のコマンドを実行してDockerコンテナを立ち上げる

cd ../
docker compose up --build

これでフロントエンドとバックエンドのアプリケーションがコンテナで動き出す。

＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊

⑤動作確認

フロントエンド：ブラウザで http://localhost:3000 を開きます。
バックエンド：APIエンドポイントは http://localhost:5000 です。

＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊＊

上長へリモートリポジトリへのpush権限を申請する
